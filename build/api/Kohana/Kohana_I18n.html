Class Kohana_I18n

Description
Description | Descendents | Vars (details) | Methods (details)

Internationalization (i18n) class. Provides language loading and translation methods without dependencies on [gettext](http://php.net/gettext).

Typically this class would never be used directly, but used via the __() function, which loads the message and replaces parameters:

// Display a translated message echo __('Hello, world');

// With parameter replacement echo __('Hello, :user', array(':user' => $username));

[!!] The __() function is declared in `SYSPATH/base.php`.
