		static method get (line 39)
	
 
	
	

Returns the output of a remote URL. Any [curl option](http://php.net/curl_setopt)  may be used.


// Do a simple GET request      $data = Remote::get($url);
// Do a POST request      $data = Remote::get($url, array(          CURLOPT_POST       => TRUE,          CURLOPT_POSTFIELDS => http_build_query($array),      ));
