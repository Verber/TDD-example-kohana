Class Kohana_Upload

Description
Description | Descendents | Vars (details) | Methods (details)

Upload helper class for working with uploaded files and [Validate].

$array = Validate::factory($_FILES);

[!!] Remember to define your form with "enctype=multipart/form-data" or file uploading will not work!

The following configuration properties can be set:

  • [Upload::$remove_spaces]
  • [Upload::$default_directory]

		static method size (line 168)
	
 
	
	

Validation rule to test if an uploaded file is allowed by file size.


File sizes are defined as: SB, where S is the size (1, 15, 300, etc) and  B is the byte modifier: (B)ytes, (K)ilobytes, (M)egabytes, (G)igabytes.
$array->rule('file', 'Upload::size', array('1M'))
