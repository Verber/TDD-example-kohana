Class Kohana_View

Description
Description | Descendents | Vars (details) | Methods (details)

Acts as an object wrapper for HTML pages with embedded PHP, called "views".

Variables can be assigned with the view object and referenced locally within the view.

		bind (line 309)
	
 
	
	

Assigns a value by reference. The benefit of binding is that values can


be altered without re-setting them. It is also possible to bind variables  before they have values. Assigned values will be available as a  variable within the view file:
// This reference can be accessed as $ref within the view      $view->bind('ref', $bar);

		render (line 330)
	
 
	
	

Renders the view object to a string. Global and local data are merged  and extracted to create local variables within the view file.


$output = $view->render();
[!!] Global variables with the same key name as local variables will be  overwritten by the local variable.

		set (line 279)
	
 
	
	

Assigns a variable by name. Assigned values will be available as a  variable within the view file:


// This value can be accessed as $foo within the view      $view->set('foo', 'my value');
You can also use an array to set several values at once:
// Create the values $food and $beverage in the view      $view->set(array('food' => 'bread', 'beverage' => 'water'));

		__get (line 159)
	
 
	
	

Magic method, searches for the given variable and returns its value.


Local variables will be returned before global variables.
$value = $view->foo;
[!!] If the variable has not yet been set, an exception will be thrown.
